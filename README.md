## Technologies:
> 1. React Hooks
> 2. Redux Hooks
> 3. JSON Server
> 4. Bootstrap

## Instructions:

> 1. `Clone the Repo`
> 
> 2. `npm install`
> 
> 3. `npm install -g json-server`
> 
> 4. `json-server --watch db.json --port 3001` to start the server
> 
> 5. Open a new window in the terminal, run `npm start`
